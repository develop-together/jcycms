# Jcy博客
这是用微信小程序做的一个自己的博客, 喜欢记得star哟！

长期维护版本，欢迎大家踊跃提交贡献代码，


## 博客主要分为三大模块： 文章/源码/我

- 文章：主要记录我在开发中遇到的问题，并且记录下来，作为技术分享

- 源码 主要记录我做的一些实用的代码，想把有些能展示的代码直接用小程序提现（比如：抽奖）

- 我：主要分为4大模块，个人基本资料、阅读历史、遗忘曲线、关于作者。
